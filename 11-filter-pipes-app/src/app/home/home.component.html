CUSTOM FILTER PIPES

PIPE is featue and it provides facility to tranform the output data into another form.
We can develope our own pipe with custom logic for tranformation and filter.
For Example: We have a list with Id, ProductName, Price column and we need to filter record based on product name then we can create custom pipe to filter the record.
