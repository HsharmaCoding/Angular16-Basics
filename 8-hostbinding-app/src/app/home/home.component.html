HostListener & Hostbinding

If we want to capture any event of host element then we should use hostlistener. It helps to listen and handle events of host element.
For example: We want to perform some events (MouseOver/Click/MouseDown/MouseUp etc) on perticular div then we can use hostlistener to listen perticular event.
We can use hostbinding in our host element to bind bind any property and attribute.
If we use hostbinding then we don't need to use renderer to access the element.
If we are not using hostbinding then we need to use renderer to access the element with hostlistener.
