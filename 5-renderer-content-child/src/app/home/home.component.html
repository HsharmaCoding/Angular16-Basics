Renderer & ContentChild

Renderer: Renderer is a inbuilt class of angular. We can modify content of dom element with the help of renderer. Renderer provides safe way to access the dom element
ContentChild: If we are projecting some information from parent component to child component, then we can access this dom element with the help of ContentChild. With the help of ContentChild we can access/update this dom element in child component.
